# PROJE: Öğrenci Sınav Analiz Sistemi (Console App)

## 🎯 Amaç

- Öğrencilerin notlarını tut
- Ortalama hesapla
- Geçti/Kaldı analizi yap
- En başarılı ve en başarısız öğrenciyi bul

# PROJE MANTIĞI

Program şunları yapacak:

1. Öğrencileri ve notları tutacak
2. Ortalama hesaplayacak
3. Geçme durumunu belirleyecek
4. En yüksek & en düşük notu bulacak

🛠️ Kullanılan Teknolojiler
Dil: C#
IDE: Visual Studio Code
Framework: .NET 10
